# Project Rules

## UI Design — Stitch First

Before implementing any UI change, new page, or visual component, always use the Stitch MCP to design it first.

- Use `mcp__stitch__edit_screens` to iterate on existing screens
- Use `mcp__stitch__generate_screen_from_text` for new screens
- Stitch project ID: `7116678432859564628`
- Present the generated screenshot to the user and get approval before writing any code

Only proceed to implementation once the design is confirmed.
